# Typescritp Api

Simple api in typescript with CQRS and DDD.

#### Install

```
$ yarn install
```

#### Start

```
$ yarn start
URL defualt api - http://localhost:3216/hello/world
```

### Cli-commands

Start tests:
```sh
$ yarn tests
```
